Types of Data

  • Categorical
    • Nominal

      This is unordered categorical data. This type of a variable may be “label-coded” in numeric form but these numerical values have no mathematical interpretation.

      • Dichotomous

        A dichotomous is a type of nominal data that can only have two possible values. These are also sometimes referred to as binary or Boolean variables.

    • Ordinal

      This is ordered categorical data in which there is strict order for comparing the values.

  • Quantitative
    • Interval

      This is a variable in which the interval between values has meaning and there is no true zero value. The addition operation between values is well defined. For example, the Celsius temperature scale is such a variable.

    • Ratio This is a variable in which the ratio between values is meaningful. It might have a true zero value that represents the total absence of the variable being measured. The multiplication operation between values is well defined. For example, the Kelvin temperature scale is such a variable. It makes sense to say 100K100\text{K}100K﻿ is twice as hot as 50K50 \text{K}50K﻿ because it represents twice as much thermal energy.

Data Cleaning

Real-world data is often “dirty”. The typical cleaning steps are:

  • type and name conversion
  • filtering of missing data
  • unifying semantic representations
  • matching of entries
  • handling of dates and time

Approaches

  • Data cleaning tools For example, Open Refine is an open-source data cleaning software.
  • Jupyter notebook with Python
