Introduction

What is a Data Scientist?

Data scientists help organizations:

  • understand their data
  • ask meaningful questions
  • derive transformative insights
  • empirically grounded decisions

What is Big Data?

Three Vs

  • Volume
    • The number of samples
    • The size of files
  • Variety
    • Structured data (.e.g. CSV)
    • Semi-structured data (e.g. JSON)
    • Unstructured data (e.g. text, e-mails, images, video)
  • Velocity
    • Conventional scientific research involves months to gather data from mere 100s cases, weeks to analyze the data and years to publish.
    • Now, for example, Twitter has on average 6000 tweets per second, 500 million per day or 200 billion per year.

Data scientists need to deal with big data. We focus on key technologies for large-scale data science

  • data models
  • processing abstractions
  • data scalability, big data
