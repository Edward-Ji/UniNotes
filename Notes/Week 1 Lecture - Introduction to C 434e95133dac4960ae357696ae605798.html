Memory - addresses and values

  • The address is the location in memory of some value.
  • Values are an arbitrary number of bits. The first bit of a value is the address it is stored at.

Computers are constantly copying values from one address to another.

main memory ↔ system memory ↔ bus (hardware) ↔ microcontroller (CPU) ↔ chip(s)

C is heterogeneous while Python is homogenous.

This means that C will behave varyingly on different hardware and operating systems while Python behaves similarly independent of those factors.

The C Programming Language

History of C

Unix was written in low-level PDP-7 assembly.

Ken Thompson invented B based on BCPL to overcome issues of PDP-7 assembly.

Dennis Ritchie built on B and called his language C.

Unix on PDP-11 was rewritten in C.

First Unix kernel in C in the year 1973

Kernighan and Ritchie published the book The C Programming Language in 1978.

Later C was standardized

  • C89 Standard (ANSI-C) introduced better parameter handling and library standards.
  • C90 standard includes a few minor modifications.
  • C99
  • C11

Features and Applications

Mainly used for

  • System software (OS, embedded systems, etc.)
  • Software that needs hardware interaction
  • Application programming, science/engineering, etc.
  • High-Performance Computing (HPC)

C

  • Is closer to the underlying machine
  • Has simple memory model
  • Assumes that programmers know best

C-compilers exist for nearly all computer architectures.

Single-board computing (e.g. Arduino) uses a compiler written in C.

C does not have object-oriented programming features like:

  • Object and classes
  • Templates
  • Operator and function overloading

Examples

  • Operating System Linux in C

    Started in 1991 by Linus Torvalds.

    Stands for Linus’ UNIX.

  • Python is written in C

The Language

C-compilers employ two languages:

  • Preprocessing language: text-macro language
  • Actual C-language: high-level programming language

Preprocessing language: text → text

  • Text-macro language
  • Definition of macro
  • Include files
  • Conditional compilation

C same as other high-level languages has

  • Control flow structures
  • References are called pointers
  • No garbage collection

    The programmer is responsible for the allocation and freeing of memory

A C program consists of a set of files containing:

  • Global variables
  • Function definitions
    • main()
    • With local variables
#include <stdio.h>

int main(int argc, char **argv)
{
	printf("Hello world!\n");
	return 0;
}

#include instructs the compiler to replace the line with the text in the specified file.
